# Plataforma Académica Básica

## Descripción del Proyecto
Este proyecto es una plataforma académica básica creada como parte de un taller para demostrar el flujo de trabajo con Git y GitHub. El objetivo es poner en práctica conceptos fundamentales como la creación de ramas, commits descriptivos, pull requests y fusión de cambios.

## Integrantes
- rdberdugoj-dot (Responsable del repositorio)

## Estructura del Proyecto
- `index.html` - Estructura HTML de la plataforma
- `style.css` - Estilos CSS de la plataforma
- `script.js` - Funcionalidades JavaScript (opcional)

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript (opcional)

## Objetivo del Taller
Aplicar el flujo básico de trabajo con Git y GitHub mediante:
- Creación de repositorio
- Uso de ramas (feature branches)
- Registro de commits descriptivos
- Carga de cambios al repositorio remoto
- Creación y revisión de pull requests
- Fusión colaborativa de código

## Cómo Usar Este Repositorio
1. Clonar el repositorio: `git clone https://github.com/rdberdugoj-dot/plataforma-academica.git`
2. Crear una rama de desarrollo: `git checkout -b feature/nombre-feature`
3. Realizar cambios y hacer commits
4. Hacer push a la rama remota
5. Crear un Pull Request
6. Esperar revisión y aprobación
7. Fusionar cambios con main
